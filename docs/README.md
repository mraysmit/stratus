# Documentation

- `architecture/` contains the system architecture and enduring design constraints.
- `decisions/` contains architecture decision records.
- `implementation/` contains phase and implementation-sequence documents.
- `operations/` contains operational acceptance documents.
- `reference/` contains stable repository and platform references.

Implementation sequence names may use phase or increment numbers. Runtime artifacts, Java packages, Maven modules, images, and deployment paths must use stable capability names instead.
